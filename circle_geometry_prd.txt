1. EXECUTIVE SUMMARY

This document defines product requirements for the "Circle" interactive
lesson module, delivered as Lesson 6.1 within Intellia SG's Grade 1 Math
program. The module targets Singapore Primary 1 students aged 6–7 and
teaches the recognition, properties, and real-world identification of
circles, fully aligned with the Singapore MOE Mathematics Framework and
the Primary 1 Targeting Mathematics syllabus on Geometry — Shapes.

The product is a standalone web page hosted at:

  https://intelliasg.com/courses/grade-1-math/lessons/circle/

It is built using React (Vite + JSX, JavaScript/CSS) and designed to
strictly mirror the visual and UX structure established at
https://wordproblem-sub.vercel.app/ and the repository
https://github.com/dsamyak/wordproblem_sub.

The module follows Intellia's 5-phase learner journey:

  Phase 1 — WONDER     → curiosity hook
  Phase 2 — STORY      → narrative-based concept introduction
  Phase 3 — SIMULATE   → sandbox-style interactive simulation
  Phase 4 — PLAY       → IntelliPlay™ gamified practice
  Phase 5 — REFLECT    → journal / LearnFlow AI prompt

Audio narration uses ElevenLabs exclusively (Voice: Alice, en-SG),
following the architecture defined in audio_generation_pipeline.md:
pre-generated .mp3 files for all scripted phase narrations,
dynamic generation via VITE_ELEVENLABS_API_KEY for runtime content,
and zero fallback to the browser Web Speech API to preserve a
consistent, premium audio experience.

4. CURRICULUM ALIGNMENT — Singapore MOE Primary 1

  Topic:   Circle (Lesson 6.1)
  Programme: Intellia SG Grade 1 Math — Section 6: Geometry (Shapes)
  Lesson URL: https://intelliasg.com/courses/grade-1-math/lessons/circle/

Source Textbooks Referenced:

  - Targeting Mathematics Primary 1A (Casco Publications)
    → Chapter on Shapes:
      • Recognising Common 2D Shapes
      • Identifying Circles in the Environment
      • Describing Shapes by Properties (curved side, no corners)
      • Sorting and Classifying Shapes

  - MOE Primary 1 Mathematics Syllabus
    → Strand: Geometry
      • Shapes and Patterns
      • Identifying and naming 2D shapes: circle, square, rectangle,
        triangle
      • Describing and classifying shapes by their properties

MOE Learning Objectives Covered:

  LO1: Name the shape "circle" correctly
  LO2: Identify circles in the environment (coins, wheels, clocks, plates)
  LO3: Describe a circle by its property: one curved side, no corners
  LO4: Distinguish circles from other 2D shapes (square, rectangle,
       triangle)
  LO5: Sort and group circles among a mixed set of shapes
  LO6: Identify circles in composite pictures and patterns

Singapore Syllabus Concepts (CPA Progression):

  Concrete   → physical circular objects (simulated digitally:
               rotating coins, rolling wheels, spinning tops)
  Pictorial  → circle diagram, shape cards, visual matching
  Abstract   → naming, property description, classification rules

Shape Properties Covered:

  Circle:
    - Exactly 1 side (curved, closed)
    - 0 corners (vertices)
    - All points on boundary equidistant from centre (not tested
      formally at P1 — introduced as "perfectly round")
    - No flat edges

Contrast Shapes Used in Practice:

  Square (4 straight sides, 4 corners)
  Rectangle (4 straight sides, 4 corners, 2 pairs equal)
  Triangle (3 straight sides, 3 corners)
  Semi-circle (introduced as "half a circle" — bonus extension)

6.3 Pre-Generated Scripts (offline, stored as .mp3 in public/assets/audio/)

  All phase narrations (Wonder, Story, Simulate instructions, Reflect
  prompts) are pre-generated via scripts/generate_audio.js and stored
  as static assets. The audioMap.js module maps each exact text string
  to its .mp3 path for zero-latency playback.

  Dynamic generation (on-the-fly ElevenLabs API call) is used for:
    - Randomised question read-alouds (play phase)
    - Praise variants ("Fantastic!", "You found it!", "Brilliant!")
    - Hint narrations for dynamically generated questions

6.4 Narration Segments (src/utils/narration.js)

  All spoken text is authored using semantic helpers:
    say("...")        → statement style
    ask("...")        → question style
    cheer("...")      → celebration style
    emphasize("...")  → emphasis style
    think("...")      → thinking style
    celebrate("...")  → celebration style

  1:1 strict parity is enforced: every word spoken must match
  exactly what appears on screen to prevent confusion for
  young learners.

6.5 Preloading Strategy

  While segment i plays, getAudioUrl is called preemptively for
  segment i+1 so the next .mp3 is already downloaded when needed,
  guaranteeing seamless narration with no latency gaps.

9. PHASE 3 — SIMULATION DESIGN (Detail)

9.1 Station A — "Draw a Circle" (Concrete)

  Visual: A dotted circular guideline on a canvas
  Interaction:
    - Child traces a finger (touch) or mouse along the circular path
    - A coloured trail follows their movement
    - Accuracy measured by proximity to the guideline (±15% tolerance)
    - On completion: circle "lights up" and bounces with a ✅
    - Mascot cheers: "You drew a perfect circle! Round and smooth!"

  Educational goal: Build kinaesthetic understanding that a circle
  has no corners — the path never "turns sharply"

  Controls: "Try Again" (resets trail) | "Done ✓" (submits attempt)

9.2 Station B — "Shape Sorter" (Pictorial)

  Visual: Two sorting bins on screen
    Bin A: "Circle" (blue bin with ⭕ icon)
    Bin B: "Not a Circle" (red bin with ✕ icon)

  Shape cards presented one at a time (10 cards total):
    Round 1 (cards 1–3):   Very clear circles vs squares
    Round 2 (cards 4–7):   Circles vs rectangles and triangles
    Round 3 (cards 8–10):  Ovals, ellipses (NOT circles), semi-circles
                            → Extension: "Is an oval a circle? No!"

  Interaction:
    - Drag-and-drop shape card into correct bin
    - OR tap card → tap correct bin (accessibility fallback)
    - Wrong sort: gentle shake + "Look carefully — does it have corners?"
    - Correct sort: pop animation + "That's right! A circle!"

  Score gate: 8/10 correct to advance to Station C (or auto-advance
  after 3 failed attempts on any card with explanation shown)

9.3 Station C — "Spot the Circle" (Abstract-Visual)

  Visual: A richly illustrated Singapore classroom scene containing
  12–15 objects. Some are circular, some are not.

  Circular objects hidden in scene:
    Clock on wall, door knob, cup opening (top view),
    coin on desk, ball on floor, wheel on toy car

  Non-circular objects (distractors):
    Book, window (rectangle), chalkboard, ruler, triangle-shaped sign

  Interaction:
    - Child taps/clicks on objects to identify circles
    - Correct tap: object glows blue + 🎉 pop
    - Wrong tap: object briefly turns red + "That has corners!"
    - Counter: "You found [X] circles! Keep looking!"

  Completion: Find all 6 circles → celebration animation → exit station

10. PHASE 4 — QUESTION BANK (100 Randomised Questions)

10.1 Question Types (10 types × 10 questions = 100)

  Type | Description                             | Example
  -----|------------------------------------------|------------------------------------
  Q1   | "Is this a circle?" — Yes/No            | Show a circle image → Yes or No?
  Q2   | Multiple choice — name the shape        | [Image of circle] → What shape is this?
  Q3   | Property question                       | How many corners does a circle have?
  Q4   | "Which one is a circle?" — pick from 4  | 4 shapes shown, pick the circle
  Q5   | Real-world identification               | Which object is shaped like a circle?
  Q6   | Sort and count                          | How many circles are in this group?
  Q7   | True/False — property statement        | "A circle has 3 corners" → True or False?
  Q8   | Spot the odd one out                   | 3 circles + 1 square → which is different?
  Q9   | Pattern completion                      | ○ □ ○ □ ○ ___ → what comes next?
  Q10  | Word problem — shape in context        | "Siti draws a shape with no corners. What
       |                                         |  shape did she draw?"

  Q4   |  10   | 4 / 4 / 2

  Q6   |  10   | 3 / 4 / 3

  Q10  |  10   | 3 / 4 / 3

10.3 Difficulty Definitions

  Easy   (Difficulty 1): Circles clearly distinguishable; no distractors
                         of similar shape; property questions are direct
  Medium (Difficulty 2): Distractors include ovals/ellipses; word
                         problems with one step; pattern with length 4
  Hard   (Difficulty 3): Composite images with overlapping shapes;
                         semi-circles included; abstract property
                         reasoning; patterns with length 6+

10.4 Singapore Context Objects Used in Questions

  Real-world circles: coin (50-cent), clock face, plate, wheel, ball,
  cookie, button, manhole cover, pizza base, fan blade, moon (full),
  bubble, eye (cartoon), round table top, hoop

  Singapore names used in word problems:
    Wei Ming, Priya, Raju, Ahmad, Mia, Jun, Siti, Ryan, Xiao Ling, Beng

11. GAMIFICATION DESIGN

11.1 Reward System

  Stars (⭐):  Earned per 10-question set (1–3 stars based on score)
  XP Points:  10 XP correct first try | 7 XP second try | 5 XP with hint
  Streak 🔥:  Fire counter for consecutive correct answers
  Streak bonus: +5 XP per correct answer when streak ≥ 5

11.2 Badges (Unlockable)

  🔵 "Circle Spotter"    — Complete Wonder + Story phases
  🏅 "Shape Sorter"      — Complete all 3 Simulate stations
  🥈 "Circle Champion"   — Score 70%+ on Play phase
  🥇 "Shape Master"      — Score 80%+ on Play phase
  💎 "Perfect Round"     — Score 10/10 in any single world
  🔥 "Streak Legend"     — Achieve a streak of 10+ correct answers
  🌟 "Full Explorer"     — Complete all 5 phases (lesson complete)
  ⭐ "Circle Hunter"     — Spot all circles in Station C first try

11.3 Feedback Mechanics

  ✅ Correct: Bounce animation + ElevenLabs celebration audio +
              "Fantastic! That's a circle! 🔵"
  ❌ Incorrect: Gentle shake + "Let's look again! Remember —
                circles have no corners."
  Hint 1: Property text highlighted + mascot gesture
  Hint 2: Visual counter animation or shape highlight
  After 2 fails: Answer revealed + animated explanation with audio

  No negative scoring — encouragement-first, always.

11.4 World Map (IntelliPlay™ Level Progression)

  World 1  — "Circle Island"    (Q 1–10:   easy identification)
  World 2  — "Coin Kingdom"     (Q 11–20:  real-world circles)
  World 3  — "Shape Safari"     (Q 21–30:  circles vs other shapes)
  World 4  — "Corner Check"     (Q 31–40:  property questions)
  World 5  — "Pattern Path"     (Q 41–50:  shape patterns)
  World 6  — "True/False Tower" (Q 51–60:  property T/F)
  World 7  — "Oddity Ocean"     (Q 61–70:  odd one out)
  World 8  — "Scene Search"     (Q 71–80:  spot circles in scene)
  World 9  — "Word World"       (Q 81–90:  word problems)
  World 10 — "Circle Palace"    (Q 91–100: mixed hard challenge)

  Each world unlocks only when ≥ 6/10 correct (1-star minimum)
  3 stars required for a bonus "hidden challenge" in each world

11.5 Mascot (LearnFlow AI Companion)

  Character:   Friendly robot (matching Intellia branding — "LearnFlow")

  Appears in:  Wonder hook, Story narration, Reflect phase
